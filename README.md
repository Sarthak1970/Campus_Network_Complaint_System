# Campus Network Complaint System

A modern, full-stack web application for students and staff to report WiFi, internet, and network-related issues on campus.

## ✨ Features

- **User-friendly Complaint Form** with validation
- **Real-time Dashboard** with statistics
- **Dark/Light Mode** support
- **Responsive Design** (Mobile + Desktop)
- **Admin Panel** (in progress)
- **Caching** with Redis for better performance

---

## 🛠 Tech Stack

### Frontend
- **Next.js 15** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **shadcn/ui** + Lucide Icons
- **next-themes** (Dark Mode)

### Backend
- **Go (Golang)**
- **Gin** (Web Framework)
- **PostgreSQL** (Supabase)
- **Redis** (Caching)
- **GORM** (ORM)

### Tools & Deployment
- Render + Vercel
- Supabase (Database)
- Upstash or Redis Cloud (Optional)
